Information Theory: A Constructive Crash Course

This text aims at intuitively explaining and proving the main definitions and theorems of Shannon's information theory with limited prerequisites; this includes: cross/conditional entropy, KL-divergence, mutual information, asymptotic codelength property (variant of AEP), source and channel coding theorems. It contains a new constructive proof of the coding theorem for discrete memoryless channel and an other non constructive proof of achievability of Shannon's capacity for the bit-flip channel.

The target audience is "undergraduates/good high school students" but anyone is welcome here.

Find the coin

As a playful formalist, I would like to introduce the subject with a small game. ;)

A coin is hidden among four opaque cups. At each round, we ask yes/no questions to uncover the cup concealing the coin. Multiple rounds are played and we aim to minimize the number of questions asked.

We first assume the coin has 111 in 444 chances to be under each cup such that the best strategy is a binary search asking 222 questions per round.

But after playing a few rounds, we notice the coin has in fact 111 in 222 chance to be in the first cup, 111 in 444 to be in the last one and 111 in 888 to be in the others. If we update our questionnaire asking: "Is the coin in the first cup? Is the coin in the last cup? Is the coin in this remaining cup?” we will ask on average only 12+14×2+(18+18)×3=1.75\frac{1}{2}+\frac{1}{4}\times2+\left(\frac{1}{8}+\frac{1}{8}\right)\times3=1.7521​+41​×2+(81​+81​)×3=1.75 questions per round.

Now imagine playing the same game using only two cups with a 70%70\%70% chance for the coin to be in the first cup and 30%30\%30% chance to be in the other. Here, we must ask one question per round.

But let's change the rules of the game allowing questions relative to two simultaneous rounds. There is a 49%49\%49% chance for the two coins to be in their first cups, 9%9\%9% chance for them to be in their last cups and 21%21\%21% for the two other cases. The best strategy will ask 12(0.49+0.21×2+(0.21+0.09)×3)=0.905\frac{1}{2}(0.49+0.21\times2+(0.21+0.09)\times3)=0.90521​(0.49+0.21×2+(0.21+0.09)×3)=0.905 questions per coin.

Entropies and KL divergence

Given nnn cups and their probabilities to contain the coin p1,…,pnp_{1},\dots,p_{n}p1​,…,pn​, as we increase the number of simultaneous rounds, the number of questions per coin of the best strategy will converge to,

−∑i=1npilog⁡2(pi)-\sum_{i=1}^{n}p_{i}\log_{2}\left(p_{i}\right)−i=1∑n​pi​log2​(pi​)

where log⁡2\log_{2}log2​ is the binary logarithm.

This is the entropy of the distribution ppp and is denoted H(p)H(p)H(p). If a single-round optimal strategy exists, we can view it as a decision tree: −log⁡2(pi)-\log_{2}\left(p_{i}\right)−log2​(pi​) is the length of the branch we will follow with probability pip_{i}pi​ when the coin is found in the ithi_{^{th}}ith​ cup.

Now if we assume the wrong distribution qqq, the number of questions we ask on average is:

−∑i=1npilog⁡2(qi)-\sum_{i=1}^{n}p_{i}\log_{2}\left(q_{i}\right)−i=1∑n​pi​log2​(qi​)

This is the cross-entropy H(p,q)H(p,q)H(p,q) and it is not symmetric. As expected:

H(p,q)≥H(p)H(p,q) \geq H(p)H(p,q)≥H(p)

The difference is called Kullback-Leibler divergence, we note:

DKL(p∥q)=H(p,q)−H(p)D_{KL}(p\Vert q)=H(p,q)-H(p)DKL​(p∥q)=H(p,q)−H(p)

In the rest of the text we will drop the 222 from log⁡2\log_{2}log2​ for simplicity.

Channel capacity

A communication channel is modeled by a source of input messages XXX transmitted and received as output messages YYY on the other end of the channel. The conditional distribution P(Y∣X)P(Y|X)P(Y∣X) defines the reliability of the channel.

Given a fixed conditional distribution P(Y∣X)P(Y|X)P(Y∣X), we can try to modify the distribution of the input messages P(X)P(X)P(X) to maximize the mutual information I(X,Y)I(X,Y)I(X,Y). This maximal mutual information is called the capacity of the channel.
